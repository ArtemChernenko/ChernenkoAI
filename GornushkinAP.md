# Краткое руководство по ~~ПШЕ~~ Git

## __Git это?__
* Git это - самая популярная система контроля версий в мире. Позволяет работать с локальными и удаленными репозиториями.

## __Основные комманды__
* *git init* - создает локальный репозиторий в папке.
* *git commit* - создает коммит (внесение изменений в файл).
* *git add* - добавляет файл (-a в git commit, -am - делает тоже самое вместе с комментарием).
* *git status* - выводит состояние репозитория.
* *git log* - выводит историю коммитов.
* *команда clear* - очищает терминал.

## Основные команды для работы с GitHub:
* git pull - забирает изменения из удаленного репозитория. Сливает их с текущей веткой.
* git push - устанавливает связь с удаленным репозиторием, передает изменения в него. Этой команде нужны права на запись. 
* git clone - создает копию удаленного репозитория на локальной машине.
* git remote - служит для управления списком удалённых репозиториев. Она позволяет сохранять длинные URL репозиториев в виде понятных коротких строк, например «origin».
